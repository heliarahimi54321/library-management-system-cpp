# Library Management System

A console-based library management system developed in C++. This project allows users to add, remove, and display books using a simple command-line interface.

## Features

* Add new books
* Display all books
* Remove books
* Simple user interaction

## Technologies Used

* C++
* Object-Oriented Programming
* STL (vector)

## Purpose

This project was built to practice C++ programming, OOP concepts, and basic data management.
